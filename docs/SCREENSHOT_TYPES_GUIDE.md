# Chrome Web Store 截图类型指南

Chrome Web Store 支持两种类型的截图，理解它们的区别很重要。

## 📸 两种截图类型

### 1. 以当地语言显示的屏幕截图（Localized Screenshots）

**定义**：针对特定语言的截图，包含该语言的界面文字。

**特点**：

- 每种语言可以上传不同的截图
- 截图中的文字是对应语言
- 用户看到的是他们语言的截图

**示例**：

- 中文截图：界面显示"敲木鱼"、"功德值"、"生命值"
- 英文截图：界面显示"Knock Fish"、"Merit"、"HP"

**何时使用**：

- 扩展有完整的多语言支持
- 界面文字较多
- 想为不同地区用户提供本地化体验

---

### 2. 全球通用的屏幕截图（Global Screenshots）

**定义**：所有语言共用的截图，不针对特定语言。

**特点**：

- 所有语言的用户看到相同的截图
- 通常使用图标、图形为主，文字较少
- 或者使用英文作为通用语言

**示例**：

- 主要展示界面布局和视觉设计
- 使用图标和图形说明功能
- 文字说明使用英文（国际通用）

**何时使用**：

- 扩展界面以图形为主，文字较少
- 不想为每种语言制作单独截图
- 想节省工作量

---

## 🎯 推荐方案

### 方案一：使用全球通用截图（推荐，省时）

**优点**：

- 只需制作一套截图
- 维护简单
- 适合图形化界面

**缺点**：

- 无法展示本地化文字
- 可能不够亲切

**适用场景**：

- 你的扩展界面以图标和图形为主
- 文字较少
- 预算或时间有限

**制作建议**：

1. 使用英文界面截图（国际通用）
2. 或者使用图形化的界面，减少文字
3. 在截图外添加英文说明文字

**示例布局**：

```
┌─────────────────────────────────────┐
│         1280 x 800 画布              │
│                                      │
│  ┌─────────────────────────┐        │
│  │                          │        │
│  │    扩展界面截图          │        │
│  │    (英文或图形化)        │        │
│  │                          │        │
│  └─────────────────────────┘        │
│                                      │
│  "Knock the wooden fish to prove    │
│   you're still alive"                │
│                                      │
└─────────────────────────────────────┘
```

---

### 方案二：使用本地化截图（更专业）

**优点**：

- 用户体验更好
- 展示完整的本地化支持
- 更容易被当地用户理解

**缺点**：

- 需要制作多套截图
- 维护工作量大
- 更新时需要同步更新所有语言

**适用场景**：

- 你的扩展有完整的多语言支持
- 界面文字较多
- 想提供最佳用户体验

**制作建议**：

1. 为每种支持的语言制作一套截图
2. 在 Chrome Web Store 后台为每种语言上传对应截图

**需要制作的截图**：

- 中文（简体）：5 张
- 英文（美国）：5 张
- 总计：10 张

## 📋 具体操作步骤

### 步骤 1：切换扩展为英文

#### 方法 A：在扩展设置中切换

1. 打开 Options 页面
2. 切换到"Settings"标签
3. 选择"English"
4. 刷新页面

#### 方法 B：修改浏览器语言

1. 打开 `chrome://settings/languages`
2. 添加"English (United States)"
3. 将其设为首选语言
4. 重启 Chrome
5. 扩展会自动使用英文

### 步骤 2：准备测试数据

使用英文测试数据：

```javascript
// 联系人示例
{
  name: "John Doe",
  email: "john@example.com",
  relationship: "Family"
}

{
  name: "Jane Smith",
  email: "jane@example.com",
  relationship: "Friend"
}
```

### 步骤 3：截取截图

按照之前的指南截取 5 张截图：

1. **screenshot-01-popup.png** - Popup 主界面
2. **screenshot-02-contacts.png** - 联系人管理
3. **screenshot-03-settings.png** - 设置页面
4. **screenshot-04-stats.png** - 统计数据
5. **screenshot-05-email.png** - 邮件预览（可选）

### 步骤 4：上传到 Chrome Web Store

1. 访问 Developer Dashboard
2. 在"图形资源"部分
3. 找到"全球通用的屏幕截图"
4. 上传 5 张截图
5. 保存

## 📞 常见问题

### Q: 必须提供本地化截图吗？

A: 不是必须的。全球通用截图就足够了。

### Q: 全球通用截图用什么语言？

A: 推荐使用英文，因为它是国际通用语言。

### Q: 可以混合使用吗？

A: 可以。你可以提供全球通用截图，同时为某些语言提供本地化截图。

### Q: 哪个更重要？

A: 对于首次发布，全球通用截图就足够了。本地化截图是锦上添花。

### Q: 我的扩展应该用哪种？

A: 对于"还活着吗"扩展，推荐使用全球通用截图（英文界面），因为：

- 界面以图形为主
- 文字较少
- 节省时间
