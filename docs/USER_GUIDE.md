## 🚀 快速开始

### 安装插件

1. 下载插件文件
2. 打开 Chrome 浏览器，访问 `chrome://extensions/`
3. 开启右上角的"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择插件的 `dist` 文件夹

### 首次使用

1. 点击浏览器工具栏中的插件图标
2. 看到禅意的木鱼界面
3. 点击木鱼，开始你的第一次敲击
4. 观察数据变化，感受禅意

## 🎯 功能详解

### 1. 敲木鱼

**如何敲击**：
- 点击中间的木鱼
- 每次点击都会触发动画效果
- 无次数限制，想敲多少次都可以

**视觉效果**：
- 木鱼会震动变形
- 从中心扩散冲击波
- 出现声波纹路
- 背景光晕闪烁

**数据变化**：
- 今日敲击 +1
- 总计敲击 +1
- 功德值增加（根据算法）
- 首次敲击时 HP +10

### 2. 功德值系统

功德值是一个多维度的累积系统：

#### 基础功德
- 每次敲击：+1 功德

#### 每日首次加成
- 每天第一次敲击：额外 +5 功德
- 跨天后重置，再次敲击可获得

#### 连击加成
- **触发条件**：3秒内连续敲击
- **加成规则**：每次额外 +1 功德
- **上限**：最多 +5 功德
- **示例**：
  - 第1次：1（基础）
  - 第2次（3秒内）：1 + 1 = 2
  - 第3次（3秒内）：1 + 2 = 3
  - 第4次（3秒内）：1 + 3 = 4
  - 第5次（3秒内）：1 + 4 = 5
  - 第6次（3秒内）：1 + 5 = 6（达到上限）

#### 连续天数加成
- **计算规则**：连续天数 × 0.5
- **上限**：最多 +10 功德
- **示例**：
  - 连续1天：+0.5
  - 连续2天：+1
  - 连续10天：+5
  - 连续20天：+10（达到上限）

#### 里程碑奖励
达到特定总敲击次数时，获得大额奖励：

| 敲击次数 | 奖励功德 |
|---------|---------|
| 10次    | +10     |
| 50次    | +50     |
| 100次   | +100    |
| 500次   | +500    |
| 1000次  | +1000   |
| 5000次  | +5000   |

#### 功德等级

| 功德值范围 | 等级称号 |
|-----------|---------|
| < 10      | 新手    |
| 10-99     | 初学者  |
| 100-499   | 初窥门径 |
| 500-999   | 虔诚信徒 |
| 1000-4999 | 修行有成 |
| 5000-9999 | 大德高僧 |
| ≥ 10000   | 功德圆满 |

### 3. 生命值（HP）系统

#### 初始状态
- 新用户：100 HP
- 状态：😊 存活中

#### HP 变化规则

**增加 HP**：
- 每日首次敲击：+10 HP
- 上限：100 HP

**减少 HP**：
- 每天不敲击：-10 HP
- 下限：0 HP

**状态变化**：
- HP > 0：😊 存活中
- HP = 0：💀 已往生

#### 时间计算
- 以每天 0:00 为分界点
- 跨天后首次敲击算作"每日首次"
- 连续不敲击会持续扣除 HP

#### 往生后
- 状态显示为"已往生"（💀）
- 仍可继续敲击
- 每日首次敲击仍可恢复 +10 HP
- 功德值继续累积

### 4. 连续天数

#### 计算规则
- 每天至少敲击一次
- 连续天数 +1
- 中断后重新计数

#### 显示位置
- HP卡片右上角
- 🔥 火焰图标 + 天数

#### 作用
- 影响功德值加成
- 显示你的坚持程度

## ❓ 常见问题

### Q1: 功德值会减少吗？
**A**: 不会。功德值只增不减，是永久累积的。即使 HP 降到 0，功德值也不会减少。

### Q2: 生命值降到 0 会怎样？
**A**: 状态会变为"已往生"（💀），但可以继续敲木鱼。每日首次敲击仍可恢复 +10 HP，连续几天就能复活。

### Q3: 连击加成如何触发？
**A**: 在 3 秒内连续敲击，每次可获得额外功德。第一次 +1，第二次 +2，以此类推，最多 +5。

### Q4: 里程碑奖励是一次性的吗？
**A**: 是的。每个里程碑只能获得一次奖励。例如达到 100 次敲击时获得 +100 功德，之后不会再获得。

### Q5: 连续天数中断了怎么办？
**A**: 连续天数会重新从 1 开始计算。建议每天坚持敲击，保持连续天数。

### Q6: 数据存储在哪里？
**A**: 所有数据都存储在本地（Chrome Storage），不会上传到云端，保护你的隐私。

### Q7: 可以导出数据吗？
**A**: 当前版本暂不支持数据导出，后续版本会添加此功能。

### Q8: 换电脑后数据会丢失吗？
**A**: 是的。当前版本数据仅存储在本地。后续版本会添加云端同步功能。

### Q9: 每天什么时候算作"新的一天"？
**A**: 以每天 0:00（午夜）为分界点。0:00 之后的首次敲击算作新一天的首次敲击。

### Q10: 可以一次性敲很多次吗？
**A**: 可以。没有次数限制，想敲多少次都可以。但要注意连击加成有 3 秒的时间窗口。
