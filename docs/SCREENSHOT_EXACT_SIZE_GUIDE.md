# 精确尺寸截图指南

本指南教你如何截取精确的 1280x800 或 640x400 尺寸的截图。

## 🎯 方法一：Chrome 开发者工具 + 设备模拟（推荐）

这是最精确的方法，可以直接截取指定尺寸。

### 步骤 1：打开要截图的页面

#### 截取 Popup 界面

1. 构建扩展：

   ```bash
   npm run build
   ```

2. 加载扩展到 Chrome：
   - 打开 `chrome://extensions/`
   - 开启"开发者模式"
   - 点击"加载已解压的扩展程序"
   - 选择 `dist` 文件夹

3. 在新标签页中打开 Popup：
   - 右键点击扩展图标
   - 选择"检查弹出内容"
   - 或者直接访问：`chrome-extension://[你的扩展ID]/src/popup/index.html`

#### 截取 Options 页面

1. 右键点击扩展图标 → "选项"
2. 或者访问：`chrome-extension://[你的扩展ID]/src/options/index.html`

### 步骤 2：打开开发者工具

1. 按 `F12` 或右键 → "检查"
2. 点击开发者工具左上角的"设备切换工具栏"图标（📱）
   - 或按 `Ctrl + Shift + M`（Mac: `Cmd + Shift + M`）

### 步骤 3：设置自定义设备尺寸

1. 在顶部设备选择器中，选择"响应式"（Responsive）

2. 在尺寸输入框中输入：
   - **宽度**：1280
   - **高度**：800

   或者：
   - **宽度**：640
   - **高度**：400

3. 确保缩放比例设置为 100%

### 步骤 4：调整页面内容

1. 刷新页面，确保内容正确显示
2. 如果是 Popup，可能需要调整 CSS：

   ```css
   /* 临时添加到 popup.css */
   body {
     width: 1280px !important;
     height: 800px !important;
     overflow: hidden;
   }
   ```

3. 如果是 Options 页面，确保内容在可视区域内

### 步骤 5：截图

#### 方法 A：使用 Chrome 截图功能（推荐）

1. 按 `Ctrl + Shift + P`（Mac: `Cmd + Shift + P`）打开命令面板
2. 输入 "Capture screenshot"
3. 选择 "Capture screenshot"（不是 full size）
4. 图片会自动下载，尺寸正好是 1280x800 或 640x400

#### 方法 B：使用截图扩展

1. 安装 "GoFullPage" 或 "Awesome Screenshot" 扩展
2. 设置截图尺寸为 1280x800
3. 点击截图

### 步骤 6：验证尺寸

1. 打开下载的图片
2. 右键 → "属性" → "详细信息"
3. 确认尺寸为 1280x800 或 640x400

## 🎯 方法二：截图后调整尺寸

如果方法一不适用，可以先截图再调整尺寸。

### 步骤 1：截取原始截图

使用任意截图工具截取扩展界面：

**Windows**：

```
Win + Shift + S
```

**Mac**：

```
Cmd + Shift + 4
```

### 步骤 2：使用图片编辑工具调整尺寸

#### 选项 A：使用 Windows 画图

1. 打开图片
2. 点击"调整大小"
3. 取消勾选"保持纵横比"
4. 选择"像素"
5. 输入：
   - 水平：1280
   - 垂直：800
6. 点击"确定"
7. 保存为 PNG

#### 选项 B：使用在线工具

**推荐工具**：https://www.iloveimg.com/resize-image

1. 上传图片
2. 选择"按像素调整大小"
3. 输入：1280 x 800
4. 下载调整后的图片

#### 选项 C：使用 Photoshop/GIMP

1. 打开图片
2. 图像 → 图像大小
3. 取消勾选"约束比例"
4. 输入：1280 x 800
5. 导出为 PNG

## 🎯 方法三：创建画布后放置截图（最专业）

这种方法可以让截图更美观，适合 Popup 这种小界面。

### 使用 Figma（免费，推荐）

#### 步骤 1：创建画布

1. 访问 https://www.figma.com/
2. 创建新文件
3. 按 `F` 创建 Frame
4. 在右侧属性面板设置尺寸：
   - W: 1280
   - H: 800

#### 步骤 2：设置背景

1. 选中 Frame
2. 在右侧"Fill"中设置背景色
   - 推荐：#F5F5F5（浅灰）
   - 或者：#FFFFFF（白色）
   - 或者：渐变色

#### 步骤 3：导入截图

1. 截取扩展界面（任意尺寸）
2. 拖拽到 Figma 中
3. 调整位置（居中）
4. 添加阴影效果：
   - 选中图片
   - 点击"Effects" → "+"
   - 选择"Drop Shadow"
   - 设置：
     - X: 0
     - Y: 4
     - Blur: 20
     - Color: rgba(0,0,0,0.1)

#### 步骤 4：添加说明文字（可选）

1. 按 `T` 创建文本
2. 输入功能说明（如"点击木鱼证明你还活着"）
3. 设置字体：
   - 字体：微软雅黑 或 Arial
   - 大小：32px
   - 颜色：#333333
4. 放置在顶部或底部

#### 步骤 5：导出

1. 选中 Frame
2. 在右侧点击"Export"
3. 格式选择 PNG
4. 点击"Export Frame"

### 使用 Canva（免费，更简单）

1. 访问 https://www.canva.com/
2. 创建自定义尺寸：1280 x 800
3. 选择背景模板或纯色背景
4. 上传截图
5. 调整位置和大小
6. 添加文字说明
7. 下载为 PNG

## 📐 针对不同界面的截图策略

### Popup 界面（360x600）

Popup 界面比较小，直接截图会显得空旷。

**推荐方案**：

1. 截取 Popup 原始尺寸（360x600）
2. 创建 1280x800 画布
3. 将 Popup 截图居中放置
4. 添加浅色背景
5. 添加阴影效果
6. 可选：添加功能说明文字

**Figma 布局示例**：


### Options 页面（全屏）

Options 页面是全屏的，可以直接截取。

**推荐方案**：

1. 使用方法一（Chrome 开发者工具）
2. 设置视口为 1280x800
3. 直接截图
4. 无需额外处理

## 🎨 美化技巧

### 1. 背景选择

**纯色背景**：

- 浅灰：#F5F5F5
- 白色：#FFFFFF
- 浅蓝：#F0F4F8

**渐变背景**：

```css
linear-gradient(135deg, #667eea 0%, #764ba2 100%)
linear-gradient(to right, #f8f9fa, #e9ecef)
```

### 2. 阴影效果

**柔和阴影**：

```
X: 0, Y: 4, Blur: 20, Color: rgba(0,0,0,0.1)
```

**强烈阴影**：

```
X: 0, Y: 8, Blur: 30, Color: rgba(0,0,0,0.15)
```

### 3. 文字说明

**位置**：

- 顶部：标题式说明
- 底部：补充说明
- 侧边：功能标注

**样式**：

- 字体：微软雅黑、Arial、Roboto
- 大小：24-36px
- 颜色：#333333（深灰）
- 对齐：居中

### 4. 标注元素

可以添加箭头或高亮标注关键功能：

- 箭头：指向重要按钮
- 圆圈：圈出关键区域
- 高亮：使用半透明色块

## 📋 完整工作流程示例

### 示例：截取 Popup 界面

#### 1. 准备阶段

```bash
# 构建扩展
npm run build

# 加载到 Chrome
# chrome://extensions/ → 加载已解压的扩展程序 → 选择 dist
```

#### 2. 截取原始 Popup

```
1. 点击扩展图标打开 Popup
2. 使用 Win + Shift + S 截图
3. 保存为 popup-raw.png
```

#### 3. 在 Figma 中制作

```
1. 创建 1280x800 Frame
2. 设置背景色 #F5F5F5
3. 导入 popup-raw.png
4. 居中放置
5. 添加阴影：0, 4, 20, rgba(0,0,0,0.1)
6. 添加标题："还活着吗 - 敲木鱼证明你还活着"
   - 字体：微软雅黑 Bold
   - 大小：36px
   - 颜色：#333333
   - 位置：顶部居中
7. 导出为 PNG
8. 保存为 screenshot-01-popup-1280x800.png
```

#### 4. 验证

```
1. 打开图片
2. 右键 → 属性 → 详细信息
3. 确认尺寸：1280 x 800
4. 确认格式：PNG
5. 确认文件大小：< 5MB
```

1. 确保扩展语言设置为中文
2. 按照上述方法截图
3. 保存为：`screenshot-01-zh-1280x800.png`

   ```

   设为首选语言

   ```
3. 重新截图
4. 保存为：`screenshot-01-en-1280x800.png`

## 💡 专业建议

### 1. 保持一致性

所有截图应该：

- 使用相同的背景色
- 使用相同的阴影效果
- 使用相同的文字样式
- 使用相同的布局方式

### 2. 突出重点

- 使用箭头指向关键功能
- 使用高亮标注重要区域
- 添加简短的功能说明

### 3. 真实但美观

- 使用真实的界面截图
- 使用合理的测试数据
- 避免空白或极端数据

### 4. 测试展示效果

- 在 Chrome Web Store 预览
- 在不同设备上查看
- 确保在小屏幕上也清晰

## 📞 常见问题

### Q: 为什么我的截图尺寸不对？

A: 确保：

1. 使用 Chrome DevTools 设置精确尺寸
2. 缩放比例设置为 100%
3. 使用"Capture screenshot"而不是"Capture full size screenshot"

### Q: Popup 截图太小怎么办？

A: 使用方法三，创建 1280x800 画布，将 Popup 截图居中放置，添加背景和阴影。

### Q: 如何让截图更专业？

A:

1. 使用 Figma 制作
2. 添加浅色背景
3. 添加柔和阴影
4. 添加简短说明文字
5. 保持一致的设计风格

### Q: 需要准备多少张截图？

A:

- 最少：1 张（Popup 主界面）
- 推荐：3-5 张（Popup + Options 各个标签）
- 最多：5 张（Chrome Web Store 限制）
