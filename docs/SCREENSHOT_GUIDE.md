# 截图制作指南

本指南将帮助你为 Chrome Web Store 制作高质量的截图。

## 📋 截图清单

需要准备以下截图：

- [ ] 1. Popup 主界面（敲木鱼）
- [ ] 2. 联系人管理页面
- [ ] 3. 设置页面
- [ ] 4. 统计数据页面
- [ ] 5. 邮件预览（可选）

## 🎯 截图要求

- **尺寸**：1280x800 像素（推荐）
- **格式**：PNG（推荐）或 JPEG
- **颜色**：24-bit RGB
- **背景**：干净整洁，无敏感信息
- **语言**：建议准备中英文两套

## 📸 详细步骤

### 准备工作

1. **构建扩展**：

   ```bash
   npm run build
   ```

2. **加载到 Chrome**：
   - 打开 `chrome://extensions/`
   - 开启"开发者模式"
   - 点击"加载已解压的扩展程序"
   - 选择 `dist` 文件夹

3. **准备测试数据**：
   - 敲击木鱼几次，让数据看起来真实
   - 添加 1-2 个测试联系人
   - 确保 HP 值在合理范围（如 80-100）

### 截图 1：Popup 主界面

**目标**：展示敲木鱼的核心功能

**步骤**：

1. 点击浏览器工具栏中的扩展图标
2. 等待 Popup 完全加载
3. 确保显示内容：
   - 木鱼图标居中
   - HP 条显示正常（如 HP: 90/100）
   - 功德值显示（如 功德: 125）
   - 今日敲击次数（如 今日: 5 次）
   - 连续天数（如 🔥 3 天）

**截图方法**：

**方法 1：使用 Chrome 开发者工具（推荐）**

```
1. 右键点击 Popup 窗口
2. 选择"检查"
3. 按 Ctrl + Shift + P（Mac: Cmd + Shift + P）
4. 输入 "Capture screenshot"
5. 选择 "Capture node screenshot"
6. 点击 Popup 的根元素
7. 图片自动下载
```

**方法 2：使用截图工具**

```
Windows: Win + Shift + S
Mac: Cmd + Shift + 4
选择 Popup 窗口区域
保存图片
```

**方法 3：使用扩展窗口固定大小**

```
1. 在 manifest.json 中设置固定大小：
   "default_popup": "src/popup/index.html",
   "default_width": 360,
   "default_height": 600

2. 使用全屏截图工具
3. 裁剪到 360x600
4. 调整到 1280x800（等比例缩放或添加背景）
```

**后期处理**：

```
1. 使用图片编辑工具（如 Photoshop、GIMP、Figma）
2. 调整尺寸到 1280x800
3. 可以添加：
   - 浅色背景
   - 阴影效果
   - 简短的功能说明文字
4. 保存为 PNG 格式
```

### 截图 2：联系人管理页面

```
3. 按 Ctrl + Shift + P
4. 输入 "Capture full size screenshot"
5. 图片自动下载
```

**注意事项**：

- 使用虚拟的测试数据（如 test@example.com）
- 不要使用真实的个人信息
- 确保界面完整显示

## 🎨 美化建议

### 添加背景

如果 Popup 截图太小，可以添加背景：

```
1. 创建 1280x800 的画布
2. 填充浅色背景（如 #f5f5f5）
3. 将 Popup 截图居中放置
4. 添加阴影效果
```

### 添加说明文字

可以在截图上添加简短说明：

```
位置：截图顶部或底部
字体：清晰易读（如 Arial、微软雅黑）
大小：24-32px
颜色：深色（如 #333）
内容：简短的功能说明（如"点击木鱼证明你还活着"）
```

### 使用 Figma 制作

**推荐工具**：Figma（免费）

**步骤**：

1. 创建 1280x800 的 Frame
2. 导入截图
3. 添加背景、阴影、文字
4. 导出为 PNG

**Figma 模板**：

```
Frame: 1280x800
Background: #f5f5f5
Screenshot: 居中，添加阴影
  - Shadow: 0 4px 20px rgba(0,0,0,0.1)
Title: 顶部，32px，粗体
Description: 底部，18px，常规
```

## 📏 尺寸调整

### 使用在线工具

- **TinyPNG**：https://tinypng.com/ （压缩）
- **Squoosh**：https://squoosh.app/ （调整尺寸）
- **Photopea**：https://www.photopea.com/ （在线 Photoshop）

### 使用命令行工具

**ImageMagick**（需要安装）：

```bash
# 调整尺寸
magick convert input.png -resize 1280x800 output.png

# 添加边框
magick convert input.png -bordercolor white -border 50 output.png

# 添加阴影
magick convert input.png \( +clone -background black -shadow 80x3+5+5 \) +swap -background none -layers merge +repage output.png
```

## 🌍 多语言截图

如果要提供多语言版本：

1. **切换语言**：
   - 在 Options 页面切换到英文
   - 重新截图

2. **文件组织**：

   ```
   images/
   ├── zh/
   │   ├── screenshot-01.png
   │   ├── screenshot-02.png
   │   └── ...
   └── en/
       ├── screenshot-01.png
       ├── screenshot-02.png
       └── ...
   ```

3. **上传时**：
   - 在 Chrome Web Store 中为每种语言上传对应的截图

## 💡 专业技巧

### 1. 使用真实但虚拟的数据

```
好的示例：
- 姓名：张三、李四
- 邮箱：test@example.com
- HP：85/100
- 功德：1,234

避免：
- 真实个人信息
- 0 或空数据
- 过于夸张的数字
```

### 2. 保持一致性

```
- 所有截图使用相同的背景色
- 相同的阴影效果
- 相同的文字样式
- 相同的尺寸比例
```

### 3. 突出重点

```
- 使用箭头或高亮标注关键功能
- 添加简短的功能说明
- 使用对比色突出重要元素
```

### 4. 测试不同设备

```
- 在不同分辨率的显示器上查看
- 确保在小屏幕上也清晰可读
- 检查在 Chrome Web Store 的展示效果
```
